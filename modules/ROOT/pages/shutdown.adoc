= 关机 (shutdown)

`shutdown` 端点用于关闭应用程序。

== 关闭应用程序

要关闭应用程序，可以按照以下基于 curl 的示例发送 `POST` 请求到 `/actuator/shutdown`：

[source,bash]
----
curl 'http://localhost:8080/actuator/shutdown' -i -X POST
----

请求会产生类似于以下的响应：

----
HTTP/1.1 200 OK
Content-Type: application/vnd.spring-boot.actuator.v3+json
Content-Length: 41

{
  "message" : "Shutting down, bye..."
}
----

=== 响应结构

响应包含有关关机请求结果的详细信息。下表描述了响应的结构：

[%autowidth]
|===
|路径|类型|描述
|`message`|`String`|描述请求结果的消息。
|===
