= Actuator REST API

这份 API 文档描述了 Spring Boot Actuators 的 Web 端点。

在您继续之前，您应该阅读以下主题：

* <<urls,URLs>>
* <<timestamps,Timestamps>>

NOTE: 为了获得下面记录的正确 JSON 响应，Jackson 必须可用。

[[urls]]
== URLs

默认情况下，所有 Web 端点都位于 `/actuator` 路径下，URL 形式为 `pass:[/actuator/{id}]`。 可以通过使用 `management.endpoints.web.base-path` 属性来配置 `/actuator` 基本路径，如下例所示：

[source]
----
management.endpoints.web.base-path=/manage
----

上述 `application.properties` 示例将端点 URL 的形式从 `pass:[/actuator/{id}]` 更改为 `pass:[/manage/{id}]`。 例如，`info` 端点的 URL 将变为 `/manage/info`。

[[timestamps]]
== Timestamps

所有端点消费的时间戳，无论是作为查询参数还是请求体中，都必须按照 https://en.wikipedia.org/wiki/ISO_8601[ISO 8601,window=_blank] 规范格式化为带偏移量的日期和时间。
